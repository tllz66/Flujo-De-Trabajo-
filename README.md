"# Flujo de Trabajo con Git

## Práctica realizada

En este repositorio demostré un flujo de trabajo básico:

1. Creé una rama llamada 'desarrollo'
2. Agregué cambios en esa rama
3. Subí la rama a GitHub
4. Creé un Pull Request
5. Fusione los cambios a la rama main
Este es el flujo básico que se usa en equipos para trabajar de forma ordenada." > README.md
